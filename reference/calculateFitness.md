# Internal: Function that calculates the fitness value

Function that calculates the fitness value of a model. This function
adheres to the necessary requirements for a function to be supplied to
[`ga`](https://github.com/luca-scr/GA/reference/ga.html).

## Usage

``` r
calculateFitness(
  .matrix_vector,
  .data,
  .model_org,
  .only_structural,
  .ms_criterion,
  .fbar
)
```

## Value

numeric
