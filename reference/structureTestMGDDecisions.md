# Internal: get structured cSEMTestMGD results

Convenience function to summarize the results of all tests resulting
from a call to
[`testMGD()`](https://floschuberth.github.io/cSEM/reference/testMGD.md)
in a user-friendly way.

## Usage

``` r
structureTestMGDDecisions(.object)
```

## Arguments

- .object:

  An R object of class
  [cSEMResults](https://floschuberth.github.io/cSEM/reference/csem_results.md)
  resulting from a call to
  [`csem()`](https://floschuberth.github.io/cSEM/reference/csem.md).

## Value

A data.frame.
